# 📘 Guide d'Utilisation du Studio Numcafé

## 🚪 Accès au Studio

1. Accédez à l'URL : `/studio`
2. Vous serez automatiquement redirigé vers `/login-studio`
3. Connectez-vous avec vos identifiants

**Comptes par défaut** :
- **Admin** : admin@numcafe.com / admin123
- **Writer** : writer@numcafe.com / writer123

---

## 🎯 Créer Votre Premier Article

### Étape 1 : Accéder à l'Éditeur

1. Cliquez sur **"Éditeur"** dans le menu latéral
2. OU cliquez sur **"Articles"** puis **"Créer un article"**

### Étape 2 : Remplir les Champs de Base

1. **Titre** : Entrez un titre accrocheur
   - Le slug sera généré automatiquement
   
2. **Résumé court** : Écrivez un résumé de 120-160 caractères
   - Un compteur vous indique la longueur recommandée

3. **Catégorie** : Sélectionnez parmi les 8 catégories disponibles

4. **Auteur** : Choisissez l'auteur (vous par défaut)

### Étape 3 : Utiliser l'Assistant IA (Optionnel)

1. Cliquez sur **"Assistant IA"** dans le header
2. Choisissez un mode :
   - **Mots-clés** : Trouvez les meilleurs mots-clés pour votre sujet
   - **Sujets** : Obtenez des suggestions de sujets à fort potentiel
   - **Plan** : Générez une structure H1/H2/H3 complète
   - **Titres** : 8 propositions de titres accrocheurs
   - **Rédaction** : Générez du contenu (choisissez le ton)
   - **Unicité** : Vérifiez l'unicité de votre contenu

3. Cliquez sur **"Générer avec l'IA"**
4. Utilisez les résultats suggérés avec les boutons d'action

### Étape 4 : Rédiger le Contenu

Utilisez la barre d'outils WYSIWYG :

**Headings** :
- H1 à H6 pour structurer votre article

**Formatage** :
- **Gras**, *Italique*, <u>Souligné</u>, Surligné
- Alignement gauche/centre/droite

**Listes** :
- Listes à puces
- Listes numérotées
- Checklists

**Éléments avancés** :
- Tableaux HTML
- Citations
- Liens (texte + URL)
- Images (upload ou collage Ctrl+V)

**Blocs promotionnels** :
1. Cliquez sur **"Bloc promo"**
2. Sélectionnez un bloc actif
3. Il sera inséré dans votre contenu

**Preview** :
- Cliquez sur l'icône "œil" pour voir le rendu

### Étape 5 : Optimiser le SEO

Remplissez la section **"Optimisation SEO"** :

1. **Mot-clé principal** : Le mot-clé cible de votre article
2. **Meta titre** : Titre optimisé pour Google (60 caractères max)
3. **Meta description** : Description pour les résultats de recherche (160 caractères max)

**Score SEO** :
- 🟢 Vert = Excellent
- 🟡 Jaune = Bon
- 🟠 Orange = Moyen
- 🔴 Rouge = Faible

Suivez les **suggestions d'amélioration** affichées.

### Étape 6 : Configurer la Publication

Dans la sidebar droite :

1. **Statut** :
   - Brouillon : Article non publié
   - En revue : Attend validation
   - Planifié : Publication programmée
   - Publié : Visible sur le site

2. **Date de publication** :
   - Laissez vide pour publication immédiate
   - OU sélectionnez date et heure pour planifier

3. **Durée de lecture** :
   - Calculée automatiquement
   - Basée sur le nombre de mots

### Étape 7 : Enregistrer

1. Cliquez sur **"Enregistrer"** dans le header
2. Votre article est créé !
3. Vous êtes redirigé vers la liste des articles

## 👥 Gérer les Auteurs

### Créer un Nouvel Auteur

1. Menu **"Auteurs"**
2. Cliquez sur **"Nouvel auteur"**
3. Remplissez le formulaire :
   - Nom complet
   - Email
   - Rôle (ex: Rédacteur, Expert SEO, etc.)
   - Bio (description complète)
   - Photo de profil (URL)
   - Réseaux sociaux (Twitter, LinkedIn, GitHub, Site web)
4. Cliquez sur **"Créer"**

### Consulter les Statistiques

Dans la liste des auteurs, vous voyez :
- Articles publiés (badge vert)
- Brouillons (badge gris)
- Vues totales
- Réseaux sociaux actifs

### Modifier un Auteur

1. Cliquez sur l'icône **"Modifier"** (crayon bleu)
2. Modifiez les informations
3. Cliquez sur **"Mettre à jour"**

---

## 🎨 Créer des Blocs Promotionnels

### Nouveau Bloc

1. Menu **"Blocs promo"**
2. Cliquez sur **"Nouveau bloc"**
3. Configurez :
   - **Titre** : Titre du bloc
   - **Contenu** : Texte promotionnel
   - **Type** : Bannière / Barre latérale / Dans le contenu / Popup
   - **Couleurs** : Fond et texte (color picker)
   - **Image** : URL optionnelle
   - **Bouton CTA** :
     - Texte du bouton
     - URL de destination
     - Couleur du bouton
4. Cochez **"Activer ce bloc immédiatement"**
5. Cliquez sur **"Créer"**

### Utiliser un Bloc dans un Article

Dans l'éditeur :
1. Positionnez le curseur où insérer le bloc
2. Cliquez sur **"Bloc promo"** dans la barre d'outils
3. Sélectionnez le bloc voulu
4. Il apparaît dans votre contenu avec le design complet

### Activer/Désactiver un Bloc

Dans la liste des blocs :
- Icône **"œil"** (vert) = Actif
- Icône **"œil barré"** (gris) = Inactif
- Cliquez pour toggle

## 🔌 Installer des Plugins

### Parcourir la Marketplace

1. Menu **"Plugins"**
2. Utilisez la barre de recherche pour trouver un plugin
3. Filtrez par catégorie :
   - SEO
   - Analytics
   - Contenu
   - Réseaux sociaux
   - Productivité
   - Monétisation

### Installer un Plugin

1. Cliquez sur **"Installer"** sur la carte du plugin
2. Le plugin est ajouté à votre liste
3. Il apparaît maintenant avec le bouton **"Activer"**

### Activer un Plugin

1. Cliquez sur **"Activer"** (bouton vert)
2. Le plugin est maintenant actif
3. Ses fonctionnalités sont disponibles

**Exemple** : Plugin "SEO Avancé Pro"
- Une fois activé, l'analyse SEO est améliorée
- Plus de suggestions dans l'éditeur

### Désactiver/Désinstaller

- **Désactiver** : Le plugin reste installé mais inactif
- **Désinstaller** : Supprime complètement le plugin

## ⚙️ Bonnes Pratiques

### Rédaction

1. **Toujours** remplir le résumé court
2. **Toujours** optimiser le SEO (mot-clé, meta)
3. Viser un score SEO **vert** (>80%)
4. Utiliser des **headings** pour structurer
5. Ajouter des **images** pertinentes
6. Insérer des **blocs promo** stratégiquement

### SEO

1. Choisir un **mot-clé principal** précis
2. L'utiliser dans :
   - Le titre
   - Le résumé
   - Les headings
   - Le contenu (naturellement)
3. Meta titre : **<60 caractères**
4. Meta description : **120-160 caractères**

### Workflow

1. **Brouillon** : Rédaction initiale
2. **En revue** : Demander validation
3. **Planifié** : Programmer la publication
4. **Publié** : Article en ligne

### Organisation

1. Utiliser les **catégories** pour organiser
2. Assigner le bon **auteur**
3. Planifier avec des **dates** précises
4. Utiliser le **calendrier éditorial** (plugin)

## 🆘 Résolution de Problèmes

### L'article ne s'enregistre pas

- Vérifiez que **tous les champs requis** sont remplis :
  - Titre *
  - Résumé *
  - Contenu *

### Le slug est incorrect

- Modifiez-le manuellement dans le champ "Slug"
- Format : `mots-separes-par-tirets`

### L'assistant IA ne génère rien

- Vérifiez votre connexion internet
- Réessayez après quelques secondes
- Note : Les données sont simulées pour l'instant

### Les images ne s'affichent pas

- Vérifiez que l'URL de l'image est valide
- Essayez de réuploader l'image
- Vérifiez les permissions du fichier

### Le plugin ne fonctionne pas

- Vérifiez qu'il est bien **activé** (pas seulement installé)
- Certains plugins nécessitent une configuration
- Rafraîchissez la page
