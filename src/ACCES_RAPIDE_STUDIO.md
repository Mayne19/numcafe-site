## ⚡ Actions Rapides

### Créer un Article
1. Menu **Articles** → **Créer un article**
2. Raccourci : Menu **Éditeur** directement

### Publier un Article
1. Éditeur → Bouton **Publier** (admin uniquement)
2. Writers : Bouton **Soumettre pour révision**

### Optimiser le SEO
1. Éditeur → Onglet **SEO**
2. Ou Menu **Assistant IA** → Onglet **Mots-clés**

### Gérer les Auteurs
1. Menu **Auteurs** → **Nouvel auteur**
2. Clic sur carte → Icône crayon pour éditer

### Installer un Plugin
1. Menu **Plugins** → Tab **Marketplace**
2. Clic sur **Installer** → **Activer**

## 🎯 Workflows Recommandés

### Workflow Rédaction Classique
```
1. Articles → Créer un article
2. Remplir titre, catégorie, résumé
3. Rédiger le contenu dans l'éditeur
4. Onglet SEO → Optimiser
5. Enregistrer brouillon ou Publier
```

### Workflow avec Assistant IA
```
1. Assistant IA → Mots-clés
2. Rechercher des mots-clés pertinents
3. Assistant IA → Sujets
4. Choisir un sujet à fort potentiel
5. Assistant IA → Plan
6. Générer une structure optimisée
7. Assistant IA → Titres
8. Sélectionner le meilleur titre
9. Articles → Créer un article
10. Rédiger avec le plan généré
11. Assistant IA → Unicité
12. Vérifier (objectif > 90%)
13. Publier
```

### Workflow de Révision (Admin)
```
1. Articles → Filtrer "En revue"
2. Clic sur article → Éditer
3. Vérifier contenu et SEO
4. Modifier si nécessaire
5. Publier ou Renvoyer en brouillon
```

## 📊 KPIs à Surveiller

### Dashboard Principal
- **Visites** : Tendance à la hausse ?
- **Taux de rebond** : < 50% = bon
- **Durée moyenne** : > 2 min = bon
- **Top 5 articles** : Quels contenus performent ?

### Dashboard SEO
- **Position moyenne** : < 20 = bon
- **CTR** : > 3% = bon
- **Impressions** : En croissance ?
- **Score SEO articles** : Majorité verts ?

### Articles
- **Ratio publiés/brouillons** : Bon flux de publication ?
- **Articles planifiés** : Pipeline rempli ?
- **Score SEO** : Majorité > 70 ?

## 💡 Astuces Pro

### Raccourcis Clavier
- `Ctrl/Cmd + S` : Enregistrer
- `Ctrl/Cmd + B` : Gras
- `Ctrl/Cmd + I` : Italique
- `Ctrl/Cmd + V` : Coller une image

### SEO
- Toujours définir un mot-clé principal
- Meta description 150-155 caractères
- Titre 50-60 caractères
- Structure H1 → H2 → H3

### Productivité
- Utiliser les brouillons pour les idées
- Planifier à l'avance
- Créer des blocs promo réutilisables
- Consulter le Dashboard SEO chaque semaine

### Organisation
- Catégoriser tous les articles
- Nommer clairement les médias
- Utiliser des tags cohérents
- Archiver les vieux articles

## 🆘 En Cas de Problème

### Article ne s'enregistre pas
- Vérifier que tous les champs obligatoires sont remplis
- Rafraîchir la page (F5)
- Vérifier le localStorage du navigateur

### Plugin ne s'active pas
- Désinstaller puis réinstaller
- Vider le cache du navigateur
- Vérifier les dépendances

### Dashboard ne charge pas
- Vérifier qu'il y a des articles publiés
- Rafraîchir la page
- Vider le cache

### IA ne fonctionne pas
- Vérifier la clé API dans Configuration
- Tester avec un autre fournisseur
- Vérifier la connexion internet
