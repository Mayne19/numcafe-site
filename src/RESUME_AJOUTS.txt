   WORKFLOW DE DÉPLOIEMENT
═══════════════════════════════════════════════════════════════

  1. TÉLÉCHARGER
     └─> Download ZIP depuis Figma Make
     
  2. EXTRAIRE
     └─> Extraire dans ~/numcafe
     
  3. GIT INIT (optionnel: test local)
     ├─> git init
     ├─> git add .
     ├─> git commit -m "Initial commit"
     ├─> git remote add origin https://github.com/USER/numcafe.git
     └─> git push -u origin main
     
  4. VERCEL DEPLOY
     ├─> Import repository
     ├─> Framework: Vite (auto-détecté)
     └─> Deploy

  ⏱️  TEMPS TOTAL: 10-15 minutes
