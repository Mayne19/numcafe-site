## 📊 Tableau de Bord

Une fois connecté, vous accédez au **Tableau de Bord** qui affiche :

### Vue d'ensemble du trafic
- **Visites** : Nombre total de visiteurs
- **Pages vues** : Nombre de pages consultées
- **Durée moyenne** : Temps passé sur le site
- **Taux de rebond** : Pourcentage de visiteurs quittant immédiatement

### Statistiques d'articles
- Articles publiés, en revue, brouillons, planifiés
- Top 5 des articles les plus consultés
- Sources de trafic (organique, social, direct, referral)

### Performance SEO
- Classement des mots-clés
- Impressions et clics Google
- CTR (Click-Through Rate)
- Évolution des positions

### Filtres de période
Cliquez sur les boutons **Jour / Mois / Année** pour ajuster la période d'analyse.

## ✍️ Créer votre premier article

### Étape 1 : Accéder à l'éditeur
1. Cliquez sur **Articles** dans le menu latéral
2. Cliquez sur le bouton **Créer un article**

### Étape 2 : Remplir les informations de base
- **Titre** : Votre titre accrocheur (génère automatiquement le slug)
- **Slug** : URL de l'article (modifiable)
- **Catégorie** : Choisissez parmi :
  - Intelligence artificielle
  - Développement web
  - Design UX/UI
  - SEO & Référencement
  - E-commerce & Marketing
  - Réseaux sociaux
  - Culture digitale
  - Productivité café
- **Résumé** : Courte description (150-160 caractères)
- **Auteur** : Sélectionnez l'auteur
- **Date de publication** : Pour planifier l'article

### Étape 3 : Rédiger le contenu

#### Barre d'outils de l'éditeur
- **H1-H6** : Titres de différents niveaux
- **B** : Gras
- **I** : Italique
- **"** : Citation
- **Liste** : Liste à puces
- **Numérotée** : Liste numérotée
- **Lien** : Ajouter un lien
- **Image** : Insérer une image
- **Code** : Bloc de code

#### Astuce : Collage d'images
Vous pouvez **coller des images directement** depuis votre presse-papiers (Ctrl+V / Cmd+V).

### Étape 4 : Optimiser le SEO
Cliquez sur l'onglet **SEO** pour :
- Définir le **Meta Title** (60 caractères max)
- Rédiger la **Meta Description** (155 caractères max)
- Choisir une **OG Image** pour les réseaux sociaux
- Définir le **mot-clé principal**
- Consulter le **score SEO** (vert = excellent, rouge = à améliorer)

### Étape 5 : Enregistrer
- **Enregistrer brouillon** : Sauvegarde sans publier
- **Soumettre pour révision** : Envoie à l'admin (rôle Writer)
- **Programmer** : Planifie la publication
- **Publier** : Met l'article en ligne immédiatement (admin uniquement)

## 🤖 Utiliser l'Assistant IA

L'Assistant IA vous aide à créer du contenu optimisé. Accédez-y via **Assistant IA** dans le menu.

### Configuration initiale
1. Cliquez sur l'onglet **Configuration**
2. Choisissez votre fournisseur : ChatGPT, Claude ou Perplexity
3. Entrez votre **clé API**
4. Cochez **Activer l'assistant IA**
5. Cliquez sur **Enregistrer**

### Rechercher des mots-clés
1. Allez dans l'onglet **Mots-clés**
2. Entrez votre **sujet** (ex: "intelligence artificielle")
3. Sélectionnez la **catégorie**
4. Cliquez sur **Rechercher**

Vous obtenez :
- Volume de recherche mensuel
- Difficulté SEO (0-100)
- CPC estimé
- Score de pertinence

### Trouver des idées de sujets
1. Onglet **Sujets**
2. Choisissez une catégorie
3. Cliquez sur **Générer des idées**

L'IA vous propose :
- Titres optimisés
- Descriptions
- Score de potentiel
- Niveau de compétition

### Générer un plan d'article
1. Onglet **Plan**
2. Entrez le **sujet** et le **mot-clé**
3. Choisissez le **ton** (professionnel, conversationnel, etc.)
4. Cliquez sur **Générer un plan**

Vous obtenez une structure H1/H2/H3 complète basée sur les meilleurs résultats Google.

### Créer des titres optimisés
1. Onglet **Titres**
2. Entrez sujet et mot-clé
3. Cliquez sur **Générer des titres**

10 suggestions vous sont proposées avec leur score de performance.

### Vérifier l'unicité
1. Rédigez votre article
2. Onglet **Unicité**
3. Cliquez sur **Vérifier l'unicité**

Objectif : **> 90% d'unicité**

## 👥 Gérer les auteurs

Accédez à **Auteurs** dans le menu pour créer et gérer les profils.

### Créer un nouvel auteur
1. Cliquez sur **Nouvel auteur**
2. Remplissez :
   - Nom complet
   - Email
   - Rôle / Titre (ex: "Expert SEO")
   - Biographie (complète et engageante)
3. Ajoutez les **réseaux sociaux** (optionnel) :
   - Twitter
   - LinkedIn
   - GitHub
   - Site web personnel
4. Cliquez sur **Créer l'auteur**

### Modifier un auteur
- Cliquez sur l'icône **crayon** sur la carte de l'auteur
- Modifiez les informations
- Cliquez sur **Mettre à jour**

### Statistiques des auteurs
Chaque carte affiche :
- Nombre total d'articles
- Total des vues
- Liens vers les réseaux sociaux

## 🔌 Installer des plugins

Les plugins étendent les fonctionnalités du Studio. Accédez à **Plugins** dans le menu.

### Vue Marketplace
Liste tous les plugins disponibles avec :
- Nom et description
- Note et nombre de téléchargements
- Catégorie (SEO, Analytics, Contenu, etc.)
- Prix (la plupart sont gratuits)

### Installer un plugin
1. Trouvez le plugin souhaité
2. Cliquez sur **Installer**
3. Une fois installé, cliquez sur **Activer**

### Plugins recommandés

#### 🔍 SEO avancé
- Analyse en temps réel
- Score coloré
- Optimisation balises meta
- **Recommandé pour tous**

#### 📊 Analytics Pro
- Intégration Google Analytics
- Suivi Search Console
- Rapports automatisés

#### ✨ Assistant IA rédaction
- Génération de contenu
- Optimisation SEO automatique
- Vérification plagiat

#### ⏱️ Durée de lecture
- Calcul automatique
- Affichage sur les articles

#### 🖼️ Optimiseur d'images
- Compression automatique
- Conversion WebP
- Améliore la vitesse

### Désactiver ou désinstaller
- **Désactiver** : Le plugin reste installé mais inactif
- **Désinstaller** : Supprime complètement le plugin

## 🎯 Optimiser le SEO

### Dashboard SEO dédié
Cliquez sur **Dashboard SEO** dans le menu pour accéder aux analyses détaillées.

#### Métriques principales
- **Impressions** : Combien de fois vos articles apparaissent dans Google
- **Clics** : Nombre de visites depuis Google
- **CTR** : Taux de clic (clics / impressions)
- **Position moyenne** : Classement moyen de vos mots-clés

#### Top mots-clés
Suivez vos 5 meilleurs mots-clés :
- Position actuelle
- Évolution (hausse/baisse)
- Impressions et clics
- CTR

#### Scores SEO des articles
Répartition de vos articles par score :
- 🟢 **Vert** : Excellent (90-100)
- 🟡 **Jaune** : Bon (70-89)
- 🟠 **Orange** : Moyen (50-69)
- 🔴 **Rouge** : Faible (0-49)

### Recommandations SEO

Le dashboard vous propose des actions à entreprendre :
- Articles sans optimisation SEO
- Mots-clés à améliorer
- Opportunités de classement

### Bonnes pratiques SEO

#### 1. Choisir le bon mot-clé
- Volume de recherche suffisant (> 100/mois)
- Difficulté adaptée à votre site (< 60 pour commencer)
- Pertinence avec votre contenu

#### 2. Optimiser le titre
- Intégrer le mot-clé principal
- 60 caractères maximum
- Accrocheur et descriptif

#### 3. Rédiger une meta description
- 155 caractères maximum
- Inclure le mot-clé
- Appel à l'action

#### 4. Structurer le contenu
- Utiliser H1 (1 seul) pour le titre
- H2 pour les sections principales
- H3 pour les sous-sections
- Intégrer naturellement le mot-clé

#### 5. Optimiser les images
- Nom de fichier descriptif
- Texte alternatif (alt text)
- Taille optimisée (< 200 Ko)

#### 6. Liens internes
- Lier vers d'autres articles pertinents
- Utiliser des ancres descriptives
- 2-5 liens internes par article

## 💡 Astuces Pro

### Workflow recommandé

1. **Recherche** : Utilisez l'IA pour trouver mots-clés et sujets
2. **Planification** : Générez un plan structuré
3. **Rédaction** : Écrivez ou utilisez l'assistance IA
4. **Optimisation** : Vérifiez le score SEO

### Organisation
- Utilisez les **brouillons** pour les articles en cours
- **Planifiez** vos publications à l'avance
- Créez des **blocs promo** réutilisables
- Consultez régulièrement la **corbeille**

## 🆘 Aide et Support

### Questions fréquentes

**Q : Je ne peux pas publier d'articles**  
R : Seuls les administrateurs peuvent publier. Les rédacteurs doivent soumettre pour révision.

**Q : Mon score SEO est rouge**  
R : Vérifiez le mot-clé principal, la meta description et la structure H1-H3.

**Q : L'Assistant IA ne fonctionne pas**  
R : Vérifiez que vous avez configuré une clé API valide dans Configuration.

**Q : Comment supprimer définitivement un article ?**  
R : Déplacez-le à la corbeille, puis videz la corbeille.

**Q : Puis-je exporter mes données ?**  
R : Actuellement, les données sont en localStorage. Migration Supabase recommandée.
